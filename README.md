# SockStream Chat
# What is it??

Behold, through a socket server running in Python, here's a simple chat that anyone can access easily on their network!

# Download + Run:
Here's how to use it:

On MAC ONLY:
Run the executable, which will run everything for you. Be patient, it may take a while!

# Try it out!

Upon running it, you'll be met with the homescreen:

There are two computers interacting locally on a network through each chat: the server, and the client.

The server hosts the chat, by clicking "Host a Local Chat" in the menu.

Once they have clicked it, they will be met with this screen.

On the client side, click "Join a Local Chat".

Once they have clicked it, they will be met with this screen:
